# vue-compo

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run dev
```

### Compiles SSR for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```


## DEPLOY (express) - AFTER build!

### Local: 
```
npm run serve
```

### Prod (on real Node server, WITHOUT any daemon): 
```
node server.js
```
